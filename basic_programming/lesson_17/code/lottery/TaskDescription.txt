Написать приложение, которое демонстрирует игру "Лотерея"

1) Для игрока выдается три лотерейных билета
2) Каждый билет содержит 5 чисел
3) Игроков может быть несколько
4) Производится розыгрыш лотереи
5) Выпадает 5 случайных значений из диапазона от 1 до 36, которые не должны повторятся

----------------------------------------------------------
Класс:
- билет (Lottery ticket)
набор значений из 5 элементов от 1 до 36 - массив из 5 элементов


- игрок (Player)
имя
набор из 3 билетов - массив


- Lottery
 -> количество игроков
 -> массив для хранения выигрышных номеров

 -> массив для хранения ВСЕХ чисел от 1 до 36 - мы его будем "перемешивать"
 после "перемешивания" мы возьмем первые 5 элементов

 -> класс PlayLottery - класс для метода main и основной логики приложения
