empty() - boolean - true если стек пустой
push(element) - помещает element в коллекцию сверху ("вталкивает")
peek() - возвращает информацию о верхнем элементе стека, но не удаляет его из коллекции
pop() - "вытаскивает" элемент с вершины стека, т.е. из самой коллекции этот элемент удаляется
search() - поиск элемента в стеке - если элемент найден, то возвращается int -
смещение от вершины стека, если не найден - то возвращается -1
