Задача 1: Работа с HashMap

Описание задачи:
Написать программу, которая позволяет пользователю управлять телефонной книгой.
Телефонная книга должна храниться в HashMap, где ключом является имя контакта, а значением — его номер телефона.
Программа должна предоставлять следующие функции:

    Добавление нового контакта
    Удаление контакта по имени
    Поиск номера телефона по имени
    Вывод всех контактов


Задача 2: Работа с HashSet

   Описание задачи:
   Написать программу, которая считывает список слов и выводит
   уникальные слова в этом списке, используя HashSet.
   Программа должна предоставлять следующие функции:

       Добавление слова в список
       Вывод всех уникальных слов

Задача 3: Работа с TreeSet

Описание задачи:
Написать программу, которая считывает список чисел и выводит их в
отсортированном порядке, используя TreeSet.
Программа должна предоставлять следующие функции:

    Добавление числа в список
    Вывод всех чисел в отсортированном порядке


Задача 4: Удаление четных чисел из списка

Описание задачи:
Написать программу, которая создает список целых чисел, а затем удаляет из него все четные числа, используя Iterator.


Задача 5: Итерация по списку строк и замена подстроки

Описание задачи:
Написать программу, которая создает список строк, а затем проходит по нему с использованием Iterator, заменяя все вхождения подстроки "Java" на "Kotlin".
