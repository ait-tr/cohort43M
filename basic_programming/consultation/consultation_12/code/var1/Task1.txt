Задача: управление списком контактов

Описание: вам необходимо реализовать систему управлением списком ваших контактов
Каждый контакт должен быть представлен в виде объекта Contact с полями name(имя) и
phoneNumber(номер телефона)
Вам нужно использовать коллекцию для хранения контактов, чтобы гарантировать
уникальность каждого контакта на основе его имени и номера телефона

Требования:
Реализовать класс Contact
реализовать класс ContactManager, который будет хранить список контактов

Класс ContactManager должен содержать следующие методы:
- addContact(Contact contact) - добавляет контакт в коллекцию
Если контакт уже существует, то выводит сообщение об ошибке

- removeContact(Contact contact) - удаляет контакт из списка
- getContacts() - возвращает список всех контактов
- getContactByName(String name) - поиск контакта по имени
- getContactByPhoneNumber(String phoneNumber) - поиск контакта по номеру телефона
