Задача:

Хранение и управление данным и об образовательных курсах

Цели:

Создать коллекцию для хранения данных о студентах
У студента имеется уникальный студенческий номер

Создать коллекцию о преподавателях
У преподавателя свой уникальный идентификатор

Создать коллекцию для хранения данных о курсах
Каждый курс связан с одним преподавателем и может иметь несколько студентов

Необходимо реализовать возможность назначить студента на курс

Entity:

Student
- String id;
- String name;
- List<Course>

Teacher
- String teacherId;
- String name;
- List<Course>

Course
- String courseId;
- String courseName;
- Teacher teacher;
- List<Student> students;


Подзадачи:

- создать коллекции для студентов, преподавателей и курсов
- Создать методы которые бы позволяли создавать :
 - преподавателя
 - студента
 - курс

 создать методы, которые бы позволяли :
 - добавить на курс преподавателя
 - добавить на курс студента

 вывод информации о курсах, студентах и преподавателях
